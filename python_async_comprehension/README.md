# Python Async Comprehension

This module demonstrates advanced async patterns including async generators and async comprehensions in Python.

## Tasks

- **0-async_generator.py**: Implements an async generator that yields 10 random numbers between 0 and 10, with a 1-second asynchronous wait between each yield.
- **1-async_comprehension.py**: Implements an async comprehension that collects numbers from the async generator using async comprehension syntax.
